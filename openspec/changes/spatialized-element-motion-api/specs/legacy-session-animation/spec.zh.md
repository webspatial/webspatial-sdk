# 旧版会话动画（历史参考）

本文件仅保留为已移除 Plan A 路径的历史背景，不再定义当前 API 的规范性要求。

## 历史摘要

- Plan A 使用 `useAnimation(config)` 加上 `enable-xr` 节点上的 `animation` prop。
- 该兼容路径已经从目标态 API 中移除。
- 新集成 MUST 使用统一的 `useAnimation(config)` timeline API 和 `xr-animation`。
